# Pizza Main Page - From Design to Code I
This project is designed to improve HTML and CSS skills as a preparation for the S8 presentation (https://github.com/doguscanavci/Pizza-Restaurant-Order-Form-Project-2) in the Workintech Full Stack Web Development program. In this project, you are expected to implement the provided design exactly using only HTML and CSS. Create the visual structure of your page by referencing the Figma design file.

**Goal:**
- To create a static HTML structure that is mobile and desktop compatible, matching the given Figma design exactly.
- To style the form and the page using CSS.
- To create a basic form structure without using JavaScript.

**Requirements:**
- Visual Studio Code
- Node.js and npm (Node Package Manager) must be installed on your computer.

#### Project Structure
- index.html: Your main HTML file.
- style.css: The CSS file containing all your styles.
- assets/: All images and other static files to be used in the project have been pre-exported for you.
- Create the HTML structure by editing the index.html file.
- Style the page and the form with CSS (style.css) by referencing the Figma design (S6 Pizza v2.1 Mobile).

#### Color Codes
1. Yellow: #FDC913
2. Light Gray: #5F5F5F
3. Dark Gray: #292929
4. Red: #CE2829
5. Beige: #FAF7F2

#### Fonts Used in Design
- The logo is pre-loaded as an SVG in the project folder.

- [Roboto Condensed](https://fonts.google.com/specimen/Roboto+Condensed)
- [Barlow](https://fonts.google.com/specimen/Barlow)
- [Quattrocento](https://fonts.google.com/specimen/Quattrocento)
- [Satisfy](https://fonts.google.com/specimen/Satisfy)

**Important Notes:**
- You will not use JavaScript in this project. The form will not be functional; it will only be developed statically.
- Aim to provide a smooth, balanced appearance on mobile and desktop screens by staying true to the Figma design.
- Pay attention to making your HTML structure semantic.
- Be careful with class naming and code organization in CSS.

**Resources:**
- Figma Design: (S6 Pizza v2.1 Mobile)
- Defining and Reusing Variables in CSS: CSS Custom Properties

## My Learning Journey & Reflection
Fidelity to Design: This project was a major exercise in "pixel-perfect" development. Replicating a Figma design exactly taught me how to measure distances and align elements professionally.

Typography Management: Working with four different fonts (Roboto Condensed, Barlow, Quattrocento, Satisfy) helped me understand how to manage font weights and imports effectively within a single project.

Form Layout Skills: Even without JavaScript, building the visual skeleton of a complex order form (sizes, toppings, notes) improved my understanding of HTML input types and CSS alignment.

## Tech Stack
- HTML5 (Semantic Elements)
- CSS3 (Custom Properties & Flexbox)
- Figma UI Design Integration
- Responsive Design

Form Düzeni Yetkinliği: JavaScript olmasa bile, karmaşık bir sipariş formunun (boyutlar, malzemeler, notlar) görsel iskeletini oluşturmak, HTML input türleri ve CSS hizalama konusundaki anlayışımı geliştirdi.
